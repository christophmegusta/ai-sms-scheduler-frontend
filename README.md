# ai sms scheduler frontend

![Alt text](screenshot.jpg "App")

## setup

install and build
```
npm install
npm run build
```

or for development
```
npm run dev
```
open browser http://localhost:5173/

remember to start main server from ai-sms-scheduler repo. see README.
otherwise data cannot be loaded.
